# ICE

ICE is a TCP/IP based Intelligent Cloud Environment library allowing for quick setup of cloud infrastructure.

## Getting Started

1) cd scripts
2) sudo chmod +x start_client_server.sh
3) ./start_client_server.sh

### Prerequisites: 

- xterm
- java jdk 
    - specifically javac
